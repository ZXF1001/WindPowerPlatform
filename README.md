# Wind Power Platform

  智慧风电场优化云计算平台的前端源码，采用Vue2构建

## 注意
  1. lat 为纬度(-90,90)，lng/lon 是经度(-180,180)

## 项目npm操作
  1. 项目依赖安装
  ```
  npm install
  ```
  ***注意：***
  ***leaflet-geotiff-plotty.js的64行开头缺一个let:***

  `(let )dataUrl = plot.colorScaleCanvas.toDataURL();`
  
  ***需要修改，否则绘制colorbar会报错***

  2. 项目运行
  ```
  npm run serve
  ```

  3. 项目编译
  ```
  npm run build
  ```
