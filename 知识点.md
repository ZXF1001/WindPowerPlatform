1.  vuex 的使用：先 npm 安装 3.0 版本，再在 src 路径下建立/store，再新建 index.js 与子模块.js，写内容导出 main.js 内引入并挂载

2. 触发的组件内写方法：
    ```
    methods: {
      handleMenu() {
        this.$store.commit('collapseMenu')
      },
    },
    ```
3. 被触发的组件内写计算属性值
    ```
    computed: {
      isCollapse() {
        return this.$store.state.tab.isCollapse
      },
    },
    ```
